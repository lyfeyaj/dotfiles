# What's Sot ?

sot is dot files and configs for myself. based on [https://github.com/ryanb/dotfiles](https://github.com/ryanb/dotfiles)

## Installation

	git clone git@github.com:sunteya/sot.git
	cd ~/.sot
	rake install
