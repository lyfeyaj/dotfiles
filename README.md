# What's Dotfiles ?

Dotfiles is dot files and configs for myself.

## Installation

	git clone git@github.com:lyfeyaj/dotfiles.git ~/.mydotfiles

	# READ ONLY
	git clone git://github.com/lyfeyaj/dotfiles.git ~/.mydotfiles
	cd ~/.mydotfiles
	rake install
