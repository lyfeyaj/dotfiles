# What's Sot ?

sot is dot files and configs for myself.

## Installation

	git clone git@github.com:sunteya/sot.git ~/.sot
	git clone git://github.com/sunteya/sot.git ~/.sot [READ-ONLY]
	cd ~/.sot
	rake install
